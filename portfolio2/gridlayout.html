Understanding Grid Layouts

The Scope

Client: College

Project Type: Editorial Design

Compentencies: Typography, Layout & Composition

Software: InDesign

The Challenge

My task for these assigments was to make thoughtful typographic choices to support readability and reflect the content. I had to select a suitable typeface, adjust size and leading for clarity, and control line length for comfortable reading. I had to use a grid to balance the text blocks with the page layout while following typographic rules and creating clear hierarchy and texture helped make the recipe and resturant menu easy to navigate.

The Solution

I approached the assignment by selecting a typeface that complements the tone of the recipe while ensuring readability. Careful attention was given to type size, leading, and line length to create a comfortable reading experience. A consistent grid system was used to support the layout and maintain balance across the page. Through clear typographic hierarchy and texture, the final design guides the reader smoothly through the content.
